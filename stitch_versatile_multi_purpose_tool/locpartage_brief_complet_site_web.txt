LOC'PARTAGE
Brief complet - Design Site Web Responsive
Maquette UI/UX web haute fidelite pour IA de design
But du fichier : Donner a une IA de design un brief complet pour generer uniquement une maquette UI/UX, sans coder le site.
Important : Le resultat attendu est un design haute fidelite, moderne, responsive et coherent, pas une application fonctionnelle.
Prompt court a copier dans une IA de design
Cree une maquette web responsive haute fidelite pour Loc'Partage, plateforme quebecoise de location de materiel entre particuliers. Ne genere pas de code ni de site fonctionnel. Cree uniquement un design UI/UX moderne, harmonieux et responsive pour mobile, tablette et desktop, avec landing page, recherche, detail annonce, publication, reservation, messagerie, dashboard utilisateur, dashboard administrateur, design system et composants reutilisables.
1. Identite et direction artistique
Identite
Nom : Loc'Partage
Concept : location de materiel entre particuliers au Quebec
Positionnement : economique, ecologique, local et communautaire
Ton visuel : moderne, clair, rassurant, premium mais accessible
Inspiration : Airbnb, Leboncoin, FatLama, Facebook Marketplace, Linear, Notion
Palette
Bleu principal #224489 pour navigation, boutons primaires, liens importants
Vert doux #8BC34A pour succes, validation, disponibilite, actions positives
Fond #F8F9FA, cartes #FFFFFF, bordures #E9ECEF
Texte principal #1F2937, texte secondaire #6B7280
Erreur #DC2626, avertissement #F59E0B, succes #16A34A
Typo
Roboto, Open Sans ou Inter
Titres bold, lisibles et hierarchises
Texte courant 14-16px mobile, 16px desktop
Boutons avec labels courts et clairs
Composants
Boutons primaires, secondaires, outline, ghost, danger, disabled
Champs : recherche, email, mot de passe, select, date picker, prix, textarea, upload photo
Cartes : annonce, categorie, utilisateur, reservation, message, statistique
Badges : Disponible, Reserve, En attente, Annule, Termine, Litige
Modals : confirmation, suppression, filtres, paiement simule
Calendrier de disponibilite, slider prix, barre de progression multi-etapes
Navigation : bottom bar mobile, header desktop, sidebar dashboard
Regles
Utiliser une grille de 8px
Coins arrondis 16-20px
Ombres legeres et realistes
Boutons faciles a toucher sur mobile
Contraste suffisant
Interface aerée avec beaucoup d'espaces blancs
Micro-interactions legeres : hover, focus, active, transition 250ms
